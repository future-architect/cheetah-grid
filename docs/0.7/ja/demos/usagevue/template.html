Usage Vue Component > Template

Template

利用可能なVue Componentのタグ名

Component Tag Name 機能 補足 LINK
<c-grid> グリッド本体 --- ---
<c-grid-column> グリッドに表示するカラム定義 --- ---
<c-grid-column-group> ヘッダを階層表示にする際のカラムグループの定義 --- ---
<c-grid-button-column> ボタンカラム定義 ボタン用カラム定義のシンタックスシュガー ---
<c-grid-input-column> 入力カラム定義 入力用カラム定義のシンタックスシュガー ---
<c-grid-menu-column> メニューカラム定義 メニュー用カラム定義のシンタックスシュガー ---
<c-grid-link-column> リンクカラム定義 リンク用カラム定義のシンタックスシュガー ---
<c-grid-icon-column> アイコンカラム定義 アイコン用カラム定義のシンタックスシュガー ---
<c-grid-percent-complete-bar-column> パーセントバーカラム定義 パーセントバー用カラム定義のシンタックスシュガー ---

vuedocも参照してください。

<template>
  <div class="demo-grid middle">
    <c-grid
      :data="records"
      :frozen-col-count="1">
      <c-grid-column
        :width="50"
        field="check"
        column-type="check"
        action="check" />
      <c-grid-column
        field="personid"
        width= "85"
      >
        ID
      </c-grid-column>
      <!-- multiple header -->
      <c-grid-column-group
        caption="Name">
        <c-grid-input-column
          field= "fname"
          width= "20%"
          min-width="150"
        >
          First Name
        </c-grid-input-column>
        <c-grid-input-column
          field="lname"
          width= "20%"
          min-width="150"
        >
          Last Name
        </c-grid-input-column>
      </c-grid-column-group>
      <!-- button -->
      <c-grid-button-column
        caption="SHOW REC"
        width="120"
        @click="onClickRecord"
      />
    </c-grid>
  </div>
</template>

<script>
/*eslint no-alert:0*/
/*global records*/
export default {
	name: 'App',
	data: {
		records
	},
	methods: {
		onClickRecord(rec) { alert(JSON.stringify(rec)); }
	}
};
</script>
