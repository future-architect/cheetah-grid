{{babel}}

ES6で記述したjsコードをES5にトランスパイルするBlock helper。

Params
  • returns {String} 
<script type="text/javascript">
//{{#babel}}
'use strict';
const str = 'handlebars';
const data = `hello ${str}!`;// template literal
const obj = {data};// object literal
const {data: res} = obj;// destructuring assignment
console.log(JSON.stringify(res));
//{{/babel}}
</script>
↓ 
<script type="text/javascript">
////babel

'use strict';

var str = 'handlebars';
var data = 'hello ' + str + '!'; // template literal
var obj = { data: data }; // object literal
var res = obj.data; // destructuring assignment

console.log(JSON.stringify(res));
//
</script>

{{wrapscript}}

jsコードをfunctionでwrapしてスコープを限定するBlock helper。

Params
  • returns {String} 
<script type="text/javascript">
'use strict';
//{{#wrapscript}}
const str = 'handlebars';
const data = `hello ${str}!`;
console.log(data);
//{{/wrapscript}}
</script>
↓ 
<script type="text/javascript">
'use strict';
////wrapscript
(function() {
//babel
'use strict';

var str = 'handlebars';
var data = 'hello ' + str + '!';
console.log(data);
//
})();
</script>

{{var}}

内容を変数に格納する

Params
  • variable name {String}
  • context {*}
  • returns {void} 
{{#var "html_varname"}}
<label style="color: red;">VAR</label>
{{/var}}
{{html_varname}}
{{{html_varname}}}
↓
<label style="color: red;">VAR</label>

{{texttrim}}

テキストの最初の空行と最後の空行を除去する。

Params
  • text {String}
  • returns {String} 
{{#var "text"}}

text　abc
text　def

text　ghi


{{/var}}
{{texttrim text}}
↓ 
text　abc
text　def

text　ghi

{{lookups}}

プロパティ値の連続参照

Params
  • object {Object}
  • ...property names {*}
  • returns {*} 
{{var "key" "b"}}
{{#json "data"}}
	{"a": {"b": {"c": 1}}}
{{/json}}
{{lookups data "a" key "c"}}
↓
1
