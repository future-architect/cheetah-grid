MultilineTextColumn

複数行テキストを表示する

プロパティcolumnTypeにmultilinetextを指定することで、
セルに複数行テキストを表示することができます。

style properties

property 説明 default
lineHeight 行に使われるスペースの量を定義します。 --
autoWrapText 自動的に折り返すかどうかを定義します。 --
lineClamp 指定した行数でテキストを切り捨てます。 --

この他に、標準のスタイルが利用できます。
